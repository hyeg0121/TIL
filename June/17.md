# 타임 리프 표현식
- ```${}``` : 변수의 값 표현식
- ```#{}``` : 속성 파일 값 표현식
- ```@{}``` : URL 표현식
- ```*{}``` : 선택한 변수의 표현식, th:object에서 선택한 객체에 접근

# 타임 리프 문법
- ```th:text``` : 텍스트를 표현할 때 사용
    - ```th:text=${person.name}``` 
- ```th:each``` : 컬렉션을 반복할 때 사용
    - ```th:each="person:${person.name}"``` 
- ```th:if``` : 컬렉션을 반복할 때 사용
    - ```th:if="${person.name}>=20"```
